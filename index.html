核心摘要

  • 项目上线7日累计新增约1,133万，累计付费流水4,253万，付费用户约106.7万
  • 7/14触底后7/15反弹为活动驱动（返利&6元礼包同日终止 + last-day效应）：单日流水从首日1,068万降至7/14的400万（-62.6%），7/15回升至471万（+17.8%）。7/15为双重重叠日——烽火江山首周返利最后一日（7/9-7/15）与6元礼包最后一日同日终止，双重last-day消费效应叠加驱动反弹。同时需注意：7/9-7/14的流水下降不完全等于"用户流失"——这段时间玩家靠免费返利就能玩，暂时不需要花钱，流水被压低了，实际付费意愿比数字看起来要强。后续需观察活动全面结束后流水能否稳住
  • 结构性分化持续加剧：ARPPU逆势上升53%（¥27→¥41），但付费渗透率从6.59%降至3.66%。留下的用户消费更强，但付费规模在萎缩
  • BP（通行证）渗透率仍然偏低：1.47%的渗透率在全系统排名倒数第二，仅触达16.6万活跃用户，与千万级用户体量严重不匹配
  • 不放回抽奖消费占比第一（39.89%），但结构脆弱：人均仅1.25-1.78抽，"一抽即走"特征突出。首充¥6送18个不放回道具（首抽耗6个/二抽耗18个），理论可两个池各抽1次——赠抽用户涌入拉低了人均数据，但也解释了1抽人群占比极高的结构成因。40%收入靠浅层消费支撑，抗风险能力弱
  • 烽火江山（放回抽奖）漏斗设计精巧，前端转化有效：登录送9抽→自费买1抽→满10再送7抽。315.2万人参与（27.8%），36.8万人付费破冰（11.7%转化率），漏斗前端运转良好。但破冰后人均仅额外追加约2.5次付费抽取，后端复购机制缺失
  • 用户抽奖深度不足，缺乏持续消费动力：烽火江山74.4%用户集中在6-10抽区间（恰好是「9赠抽+1自费」触发满10送7的位置），说明漏斗驱动了破冰行为但未能推动持续消费；不放回抽奖人均仅1.25-1.78抽，首充赠18个道具（两池各首抽6个=12个，剩6个不够任何一池二抽的18个）与CBT继承双重赠送叠加，真实付费用户的消费深度被赠抽用户大幅稀释
  • 付费分层呈断崖式衰退：微R以上各层级付费人数7日下降55-84%，且渗透率持续下滑，付费底盘在快速萎缩
  • 收入结构头部依赖严重：中大R及以上用户仅占付费人数的不足4%，却贡献了近半流水，抗风险能力弱

诊断结论：BP是目前商业化系统中最大的短板 BP渗透率仅1.47%，在6大商业化系统中排名第5（仅高于捆绑包）。消费占比13.21%虽然看起来排名靠前，但这是建立在人均¥53.0的较高客单价基础上——即少数人买了，但绝大多数活跃用户没有购买。

对于一个上线初期、用户规模超千万的项目，BP仅触达1.5%的用户，说明BP的价值传达和购买引导存在明显短板。
可能的原因分析
  • BP的奖励吸引力不足或感知门槛过高，用户认为"不值得买"
  • BP的入口曝光不足，新用户不知道有通行证系统
  • BP任务体系可能过于繁重或奖励周期过长，用户在开服初期还没感受到BP价值
  • 开服初期抽奖类活动吸引力远高于通行证，用户消费预算被抽奖分流
  • BP定价策略可能偏高，与微R用户（占付费用户约66%）的支付意愿不匹配

烽火江山：登录赠9抽→自费买1抽→满10再送7抽，是一个精心设计的递进漏斗
烽火江山的完整活动链路为：登录1天送9抽 → 自费购买第10抽 → 累计满10抽 → 再送7次烽火江山抽奖。赠抽拉新→付费破冰→满10送7奖励，递进逻辑清晰。

漏斗效果：参与率27.8%（315.2万人），付费渗透率3.25%（36.8万人付费），RMB消耗1,897.8万（消费占比28.44%）。74.4%用户集中在6-10抽区间——这恰好是「9次赠抽 + 1次自费抽取」触发满10送7的位置。这说明大量用户完成了「领赠抽→自费破冰→触发满10奖励」的完整路径，而非单纯「领完就走」。

漏斗的评价：前端设计成功，后端复购不足。自费1抽的低门槛+满10送7的额外奖励有效驱动了首次付费行为，但用户在获得额外赠送的7抽后缺乏继续投入的理由——人均扣除赠抽和奖励后额外付费约3.49次（含自费的第10抽），意味着平均每个付费用户在破冰之后仅额外追加了约2.5次付费抽取。满10送7的奖励消耗完后没有下一个目标。

补充：烽火江山首周返利活动（7/9-7/15期间）向用户发放了大量免费返利资源——玩家拿到免费资源后暂时不需要自己掏钱，主动付费意愿被"盖住"了。7/15返利与6元礼包同日终止，留存用户的付费意愿集中释放，驱动反弹。这也意味着7/9-7/14的流水下降不完全等于"用户流失"：那几天流水低，有一部分是因为免费返利替掉了本该发生的付费，实际付费意愿比账面数字更强。

烽火江山漏斗诊断：付费破冰成功，但「满10送7」奖励消耗后用户失去动力
  • 漏斗前端（赠抽→付费破冰）运转良好：仅3.2%的用户停在1-5抽（连赠抽都没领完），74.4%到达6-10抽区间——这个区间恰好覆盖了「9次赠抽 + 1次自费抽取」触发满10送7的位置。自费1抽的低门槛定价驱使用户从免费跨越到付费，设计合理
  • 满10送7奖励有效但消耗过快：累计10抽赠送的7次烽火江山抽奖本应延长用户的消费周期，但额外7抽消耗完毕后缺乏衔接的「下一个台阶」。75%的用户获得最高级物品后停止抽取，说明奖励获取→满足感→离开的链条没有被打断
  • 漏斗后端（持续复购）严重缺失：扣除赠抽和满10奖励后人均仅额外付费约3.49次（含自费的第10抽），95.0%的用户不超过20次（即额外付费不超过11次）。25%的高级物品获取率（9-12抽即可获取）反而加速了满足感——拿到好东西就停了，缺乏「下一个追逐目标」

* 涂鸦狂潮原始获取率82%（55.1万参与用户中45.3万抽1次即获得），但考虑到CBT2/CBT3期间已获得该道具的玩家公测后自动继承，1抽人群的获取主要来自CBT继承。实际需要通过抽奖获取的用户集中在6-7抽区间（878+3,605=4,483人），真实获取率约0.81%。

首充道具机制是不放回人均抽数偏低的核心结构原因 首充¥6赠送18个不放回抽奖道具（非1次抽奖）。不放回抽奖首抽消耗6个道具、第二抽消耗18个道具。这意味着首充用户两个池子各可首抽1次（绯夜猫影6个+涂鸦狂潮6个=12个，剩余6个不够任一池的第二抽）。

这个定价设计决定了不放回奖池的消费结构：大量用户首充后各池体验1抽即止——因为第二抽需要18个道具（需再充值才能凑齐），从6个跳到18个的阶梯太陡，几乎没有自然的过渡引导。首充¥6带来了庞大的"两池各1抽"用户群，但这批人统计在参与人数中、贡献了渗透率却几乎不产生额外消费，显著拉低了人均抽数。

对照烽火江山的赠抽机制（登录送9抽+满10送7抽，人均12.49抽），不放回奖池的"赠抽—付费"梯度存在根本性断层：烽火江山通过赠抽建立消费习惯后再引导付费，而不放回的首充道具机制是"给够1抽、卡住第2抽"，用户刚体验就被门槛拦下，消费链路在第二抽之前就断裂了。
不放回抽奖RMB消耗对比（7日累计）
绯夜猫影收入远超涂鸦狂潮，但两者人均消费都偏低 绯夜猫影RMB消耗2,101.1万，是涂鸦狂潮（560.3万）的3.8倍。但绯夜猫影人均消费¥23.96、涂鸦狂潮仅¥10.17，两个卡池的客单价都处于低位。绯夜猫影凭借更大的用户基数（87.7万 vs 55.1万）拉开了收入差距，但人均消费深度不足的问题在两个卡池中同样严重。
不放回抽奖人均抽数对比
不放回抽奖抽数极低，首充道具机制锁死了消费上限 绯夜猫影人均仅1.78抽（87.7万人参与）、涂鸦狂潮仅1.25抽（55.1万人参与）。首充¥6送18个道具（两池各首抽6个=12个，剩6个不够任一池二抽的18个）为两个奖池注入了大量"首充体验1抽就停"的用户——他们各池抽完首充赠送的1次就结束，进一步拉低了人均。

绯夜猫影用户抽了就走、涂鸦狂潮82%用户1抽即停（其中绝大部分为CBT继承，真实通过抽奖获取的仅约0.81%，集中在6-7抽区间）——两个极端，都缺乏复抽动力。建议关注：排除首充赠抽和CBT继承后，真实自费玩家的消费深度是多少？道具消耗阶梯（6→18）是否存在优化空间？

走势特征与归因分析
核心判断：7/14触底后7/15反弹，为活动双重叠加而非自然底部 7天流水从1,068万降至7/14的400万（-62.6%），7/15回升至471万（+17.8%），付费用户数也从7/14的10.4万回升至11.4万（+9.5%）。7/15是返利与礼包的双重终止日：（1）烽火江山首周返利最后一日（7/9-7/15）——返利期间玩家靠免费资源就够用了，付费被压住，最后一日集中释放；（2）6元礼包最后一日——礼包同日下架，低门槛付费的末班车效应进一步拉升微R/小R消费。双重last-day效应同日叠加，共同驱动了7/15的集中反弹。
结构性分化：规模萎缩 vs 质量提升
付费用户量缩质升：ARPPU从¥27.11升至¥41.38（+52.6%），平均单次付费金额从¥18.71升至¥25.10（+34.1%）。这说明留下来的付费用户消费意愿更强、单次付费更高——不是所有用户都在流失，而是低质量付费用户在离开，核心付费用户群体正在沉淀。

但ARPU整体走低后受活动拉动反弹：从¥1.79降至7/14的¥1.16（-35.2%），但7/15反弹至¥1.52（+31.0%）。ARPU=付费渗透率×ARPPU，7/15的反弹说明6元礼包有效拉升了付费渗透率（从3.02%→3.66%），但需观察促销结束后渗透率是否回落。
商业化系统层面的三大关联归因
  1. BP触达断层 → 付费转化漏斗断裂：BP渗透率仅1.47%，千万活跃用户中98.5%没有接触过这个最基础的付费入门产品。没有BP这个"付费第一站"，大量潜在付费用户直接面对抽奖和商城等中高门槛系统，付费转化路径过长。
  2. 烽火江山漏斗前端有效但后端断流 → 付费用户未转化为复购：登录赠9抽→自费买1抽→满10送7抽的漏斗设计驱动了315.2万人参与和36.8万人付费破冰（11.7%转化率）。但满10奖励消耗后没有下一个目标驱动继续消费——人均扣除赠抽和奖励后额外付费仅3.49次（含自费的第10抽），即破冰之后平均仅追加约2.5次付费。前端漏斗运转良好，但复购机制缺失。
  3. 不放回抽奖人均不足2抽 → 道具消耗阶梯断层 + 首充用户快速流失：不放回抽奖吸引了60.4万消费用户（渗透率5.33%，全系统最高），但人均仅1.25-1.78抽。核心结构原因：首充¥6送18个道具，首抽6个/二抽18个，用户两个池各体验1抽后用尽12个道具，剩余6个不够任一池的二抽——从6到18的消耗跳跃制造了"第二抽壁垒"。大量首充用户因此停在第1抽，这个入口"以低价召之即来"却因阶梯断层"留不住"。
关键观察：7/15反弹为双重叠加，非自然底部确认
反弹归因：7/15流水+17.8%、付费用户+9.5%，两项同步回升。7/15为两重终止日——（1）烽火江山首周返利（7/9-7/15）最后一日，返利末日的消费释放效应；（2）6元礼包最后一日，低门槛付费末班车效应叠加。多付费层级同步反弹（微R+14.5%、中大R+13.1%）印证了双重终末对各层级的普适拉动效果。

后续关注：返利与礼包全面结束后（7/16起）数据是否回落是关键验证。若7/16-7/17流水回落至400万以下，则说明自然底部在350-400万区间，需通过常态化运营而非一次性促销来维稳。超R用户持续流失（仅剩9人）对头部收入的长期影响仍需密切关注。

优化建议
1. BP（通行证）紧急优化 — 最高优先级
  • 降低购买门槛：考虑推出开服限时折扣或入门版BP，降低微R用户的购买阻力（微R付费用户人均仅¥7，对当前BP定价可能有价格敏感）
  • 强化BP入口曝光：在主界面、抽奖结算页、任务完成页等高频触达点增加BP购买引导
  • 前置BP价值感知：提供首日免费试玩体验或预览全部奖励，让用户在购买前就能感受到BP价值
  • 增加即时正反馈：BP前期奖励加大，缩短首次奖励获取周期，避免用户因"看不到回报"而放弃
  • 与抽奖联动：BP等级提升赠送抽奖券或咕咕币，打通BP与抽奖系统的消费链路
2. 烽火江山漏斗后端补齐 — 高优先级
  • 漏斗前端（赠抽→付费破冰）保持现状，不必大改：登录赠9抽 + 自费买1抽 + 满10送7抽的设计已被数据验证有效——27.8%参与率、11.7%破冰转化率、仅3.2%用户未领完赠抽。当前核心问题不在前端，而在满10奖励消耗之后
  • 在「满10送7」奖励消耗后增设下一个目标：满10送7的7次免费抽消耗完毕后，立即弹出「再抽5次必得限定道具」「累计20抽送XX」，避免奖励消耗后的动力真空
  • 首付破冰后设置阶梯消费引导：第10抽破冰成功后，弹出「再抽5次得限定」等阶梯目标，将用户从单次付费推向多轮消费。当前人均仅额外付费3.49次（含自费第10抽），说明破冰之后的引导断层严重
  • 利用满10送7延长用户消费周期：当前满10送7的7次抽奖是赠送的，用户消费完就走。可设计为「满10送7→消耗完7次→再抽N次触发下一个奖励」，形成持续循环
3. 不放回抽奖卡池改造 + 道具消耗阶梯优化 — 高优先级
  • 优化道具消耗阶梯（6→18）是覆盖用户最广的杠杆：首充¥6送18个道具，用户两池各抽1次后剩6个——距离第二抽的18个还差12个。这个"悬在空中的6个道具"是天然的转化引导锚点。建议：推出「6元补充包（+12个道具）」精准凑齐第二抽，或直接降低第二抽消耗至12个（或阶梯增长6→12→18），让首充用户的赠抽道具不被浪费
  • 绯夜猫影：人均1.78抽、高级物品获取率仅7%——作为该卡池的最终追求，93%的用户未达到终点就已放弃。建议降低保底抽数或提升前期物品质量，让更多用户有动力追至终点
  • 涂鸦狂潮：人均1.25抽最低，82%用户1抽即停（绝大部分为CBT继承，6-7抽区间仅4,483人、真实获取率约0.81%）。需要增加更多层级的奖励来驱动复抽
  • 整体策略：不放回抽奖的"一次性"特征太强，首充道具阶梯和CBT继承又放大了这种一次性。建议从道具消耗阶梯入手（6→12→18平滑过渡），让用户有自然的"再充一点就能再抽一次"的心理锚点，并在赠抽消耗后设置平滑的付费过渡
4. 付费用户留存与分层经营 — 长期优先级
  • 微R用户基本盘维护：微R占付费人数约66%但流失74.2%，是挽回付费规模的关键。建议推出每日小额礼包、连续登录奖励等低门槛付费点
  • 推动微R→小R升级：微R渗透率从50.81%降至16.63%，但客单价仅¥7。通过首充翻倍、限时升级礼包等手段推动微R向小R（¥30-100）转化
  • 中大R用户深度运营：中大R+大R占付费人数不足4%但贡献约49%流水，是核心收入来源。需要专属客服、VIP礼包、提前体验等差异化服务
  • 超R用户紧急挽留：超R从58人降至9人（-84.5%），需立即启动头部用户1对1触达和专属福利
  • 遏制渗透率下滑：各层级渗透率7天内全面下滑，需要通过运营活动（如周末双倍、限时返利）拉升日活和付费意愿
5. 捆绑包与商城补齐 — 低优先级
  • 捆绑包渗透率仅0.21%（约2.38万人），消费占比3.31%。四个组合包中两个上架时间标注"暂无"，另两个缺少价格信息，可能存在上架配置问题
  • 商城渗透率1.43%，消费占比仅2.95%，作为常驻商业化入口严重不足。建议优化商品结构和定价策略
总结

失控进化上线7日累计新增1,133万、付费总额4,253万，开局数据体量可观。但商业化结构存在"隐性失衡"的隐患：

BP渗透低（1.47%，倒数第二），千万活跃用户中98.5%未接触过BP；不放回抽奖"一抽即走"（人均1.25-1.78抽），39.89%的收入靠浅层消费支撑，首充¥6送18个道具（首抽6个/二抽18个）使两池各可首抽1次，但道具消耗阶梯（6→18）制造了"第二抽壁垒"，大量首充用户停在1抽无法突破；烽火江山漏斗前端有效但后端断流（登录赠9抽→自费买1抽→满10送7抽的递进设计精巧，315万人参与、36.8万人付费破冰，但满10奖励消耗后人均仅追加2.5次付费，复购机制缺失）；收入高度依赖抽奖和头部付费用户（抽奖占68%流水，中大R以上贡献约49%）。

不放回抽奖消费占比排名第一，但人均不足2抽——首充道具的6→18消耗阶梯缺陷和CBT继承双重因素叠加，使参与人数被赠抽用户大幅稀释，真实付费用户的人均抽数被显著拉低。一旦奖池吸引力下降，40%收入可能迅速蒸发。烽火江山的漏斗设计值得肯定：赠抽拉新→付费破冰→满10送7，递进逻辑清晰，但关键短板在于——满10送7的奖励消耗完后没有下一个消费目标，用户在获取奖励后直接停滞，复购动力断裂。

叠加开服脉冲效应快速消退（各层级付费人数7日下降55-84%），若不及时优化BP触达、修复不放回道具消耗阶梯、提升卡池付费转化和消费深度、加强微R留存，后续付费规模可能持续下滑。7/15的反弹由返利最后一日、6元礼包最后一日双重叠加驱动，属于活动拉动而非自然底部确认。需特别关注：7/16起活动全面结束后流水是否能稳住，以及7/9-7/14的流水下降中，免费返利压低了实际付费数据的成分。
